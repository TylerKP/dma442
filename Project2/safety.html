Note: We are not responsible for any injury or harm that may incur when using a lathe. Use your best judgement. These are only basic guidelines. Make sure to read all warnings and labels, as well as the operating guide for your lathe.

General Instructions

  1. Make sure the area around the lathe is clean and clear of sawdust or any flammable material.
  2. Use safety goggles at all times.
  3. A sweatshirt or longsleeved shirt is recommended, as are shoes, when using a lathe.
  4. Use a dust mask when operating the lathe.
  5. Before actually using the lathe, make sure that the it is centered, that there is no wobbling. This will damage your project.
  6. When using the lathe, most come with a guide. Never ever point the lathe tool down, always point the tool up. If you point the tool down, it will be taken from your hands and cause injury.
  7. When using the linseed oil, make sure you are wearing gloves. Linseed oil can cause skin irritation.

Additional Instructions When Using Acrylic

  1. If you are using acrylic, gloves are highly recommended.
  2. When sanding the acrylic, make sure you have a bowl of water on hand. Put the sandpaper on the acrylic for a few seconds, then dip the sandpaper into the water and repeat.
  3. Skip the linseed oil and wax-- the acrylic is basically plastic; once the sanding is done, all you have to do is put the project together.
